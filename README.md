# Invoice Generator

Proyek web aplikasi untuk membuat dan mengunduh invoice sebagai PDF dengan manajemen akun pengguna.
Project ini terdiri dari Frontend (React) dan Backend (Laravel).

## Struktur Project
- `frontend/`: Aplikasi React (Vite)
- `backend/`: API Server (Laravel)

## Cara Menjalankan

### Backend (Laravel)
1. Masuk ke folder backend: `cd backend`
2. Install dependencies: `composer install`
3. Copy `.env.example` ke `.env`: `cp .env.example .env`
4. Konfigurasi database di file `.env`
5. Generate key: `php artisan key:generate`
6. Jalankan migrasi database: `php artisan migrate`
7. Jalankan server: `php artisan serve`

### Frontend (React)
1. Masuk ke folder frontend: `cd frontend`
2. Install dependencies: `npm install`
3. Jalankan server development: `npm run dev`

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, html2pdf.js
- **Backend**: PHP Laravel, MySQL, Sanctum (Auth)

## Fitur Utama
- **Pembuatan Invoice**: Input data Seller, Customer, Item, dan Pajak dengan perhitungan otomatis.
- **Preview Real-time**: Melihat tampilan invoice secara langsung saat diedit.
- **PDF Export**: Unduh invoice dalam format PDF siap cetak.
- **Manajemen Kontak**: Simpan data Customer dan Seller untuk digunakan kembali.
- **Riwayat Invoice**: Simpan dan kelola riwayat invoice yang pernah dibuat.
